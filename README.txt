VoltGuard Engenharia - versão elétrica (frontend + backend)
Frontend: public/ (abra index.html ou sirva via backend)
Backend: Node.js Express (server.js) — serve a pasta public e recebe POST /api/contact
Como rodar (local):
1. Instale node.js (>=16)
2. cd backend
3. npm install
4. npm start
Teste: abra http://localhost:3000

OBS: Atualize o número do WhatsApp em public/script.js (const WHATSAPP_NUMBER) e no link do botão se desejar.
